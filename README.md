# AI智能体管理系统源码

#### 1、介绍

本项目是一个AI智能体管理平台系统，专为开发者、学习者和爱好者设计，旨在帮助用户理解和实现AI智能体与硬件设备的结合。通过该平台，用户可以创建和管理AI智能体，集成语音识别、语音合成及设备控制，探索AI技术与硬件设备之间的协作！

#### 3、应用场景

该系统适合用作：

1、学习AI智能体与硬件结合开发的基础平台

2、开发个人项目，进行智能设备管理与控制

3、学习如何在开发中使用AI语音技术（如语音识别、语音合成）

#### 4、使用技术

本项目采用了现代的技术栈，旨在保证高效开发与扩展性，适合学习者了解最新的技术应用。

1、后端技术：

①Java 8：作为项目的核心编程语言，稳定高效。

②Spring Boot 2.7：提供快速开发的框架，适合构建微服务和集成AI模块。

2、前端技术：

①Vue.js 2：流行的前端框架，适合构建响应式网页应用，提供良好的用户体验。

②Node.js 16版本：确保项目在前后端交互时的高效性。

3、AI技术集成：

①百度云智能体：为系统提供智能体管理功能。

②YUMOU AI-VOICE-Z01/ESP32-S3设备：智能硬件设备，结合语音交互功能。

③火山引擎：提供高效的语音识别与语音合成功能，支持“一句话识别”和音频生成。

#### 5、网站内容

1、智能体管理界面：用户可以方便地添加、管理和分类智能体，支持不同类型智能体的操作。

2、设备管理功能：后台集成了YUMOU语音设备（如AI-VOICE-Z01/ESP32-S3），允许管理员进行硬件设备的管理、控制及配置。

3、语音识别与合成配置：内置火山引擎的语音识别与语音合成功能，用户可以配置智能体语音交互行为。

4、管理员管理界面：为系统管理员提供集中化管理功能。

5、前台展示：在智能体应用网站上，前台会展示后台所管理的智能体，用户可以直接与智能体进行交互，体验AI语音识别和语音合成功能。（本系统接入百度智能体，前台仅提供跳转到对应智能体平台进行问答交互，不提供AI在线聊天功能）
